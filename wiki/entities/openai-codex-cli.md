# OpenAI Codex CLI

OpenAI's open-source (Apache-licensed) terminal coding agent; can target a local
OpenAI-compatible server instead of OpenAI's cloud.

**Type:** CLI · **License:** open-source, Apache (free) · **Local model:** ✅ via OpenAI-compatible base URL

## Summary
Free, open-source CLI agent from OpenAI. Although it ships pointed at OpenAI's API, its
base URL can be redirected to a local OpenAI-compatible endpoint (Ollama / vLLM), making a
free + offline setup possible (see [[local-model-coding-agent]]). Candidate CLI agent in
[[ai-coding-agents]]. Distinct from the *cloud* Codex agent inside ChatGPT.
