# Cline

Open-source VS Code extension agent with first-class local-model support; **Roo Code** is a
popular fork.

**Type:** IDE (VS Code extension) · **License:** open-source (free) · **Local model:** ✅ native (Ollama/LM Studio)

## Summary
Free, open-source editor agent (formerly "Claude Dev") that runs inside VS Code and can
drive a local [Ollama](https://ollama.com) / LM Studio model directly — a free + offline
editor option (see [[local-model-coding-agent]]). The **Roo Code** fork is equivalent for
these purposes. Listed in [[ai-coding-agents]] as an IDE candidate.
