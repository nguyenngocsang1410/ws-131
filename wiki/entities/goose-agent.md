# Goose (Block)

Block's open-source local AI agent, usable from the terminal or a desktop app, with native
support for local models.

**Type:** CLI (+desktop) · **License:** open-source (free) · **Local model:** ✅ native

## Summary
General-purpose open-source agent that runs against configurable providers including local
[Ollama](https://ollama.com) models, so it qualifies as a free + offline option (see
[[local-model-coding-agent]]). Listed among the realistic free+local candidates in
[[ai-coding-agents]].
